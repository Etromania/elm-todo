# Elm Todo

### Install:
```
git clone git@github.com:etromania/elm-todo.git
cd elm-webpack-starter
npm install
```

If you haven't done so yet, install Elm globally:
```
npm install -g elm
```

Install Elm's dependencies:
```
elm package install
```

### Serve locally:
```
npm start
```

### Build & bundle for prod:
```
npm run build
```
